
# CRUD de Clientes 📥
Este repositório contém uma aplicação web para gerenciamento de clientes, desenvolvida como parte da tarefa da disciplina de Engenharia de Software 3, ministrada pelo professor Rodrigo na Fatec de Mogi das Cruzes. O projeto consiste em um sistema completo de CRUD (Create, Read, Update, Delete) para clientes, com foco na funcionalidade de cadastro.

## Funcionalidades 🌟
- Cadastro de Clientes: Permite a entrada de dados pessoais, como nome, data de nascimento, e-mail, CPF e gênero.
- Listagem de Clientes: Exibe uma lista dos clientes cadastrados.
- Busca de Clientes: Funcionalidade para buscar clientes cadastrados.

## Estrutura do Repositório 🗂️
O projeto está dividido em duas branches:
- **front-end/develop**: Contém o código da aplicação front-end desenvolvida em React e TypeScript. Para baixar o front-end, acesse a branch front-end/develop.
- **back-end/develop**: Contém o código da aplicação back-end desenvolvida em Java utilizando o Spring Framework. Para baixar o back-end, acesse a branch back-end/develop.

## Contribuindo para o Projeto 🤝
Para fazer parte do projeto, siga as etapas abaixo:
1. Faça um Fork do Repositório: Crie uma cópia do repositório em sua conta do GitHub.
2. Crie uma Nova Branch:
  - Navegue até o repositório clonado na sua máquina local.
  - Crie uma nova branch para suas alterações. É recomendável usar um nome descritivo:
  ```bash
  - git checkout -b nome-da-sua-branch
  ```
3. Realize as Alterações: Faça as modificações desejadas no código.
4. Faça Commit das Alterações:
  - Adicione suas alterações:
  ```bash
  git add .
  ```
  - Faça commit:
  ```bash
  git commit -m 'Descrição das suas alterações'
  ```
5. Envie suas Alterações:
  front: 
  ```bash
  git push origin front-end/feature/nome-da-sua-branch
  ```
  back: 
  ```bash
  git push origin back-end/feature/nome-da-sua-branch
  ```
6. Abra um Pull Request:
  - Navegue até o repositório original no GitHub.
  - Clique na aba "Pull Requests" e em seguida clique no botão "New Pull Request".
  - Selecione sua branch e clique em "Create Pull Request". Adicione uma descrição clara sobre suas alterações.
